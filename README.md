# Immersed Releases

This repository contains release builds for the [Immersed](https://getimmersed.app) desktop application.

## About

This is a public repository that hosts release files for Immersed, allowing automatic updates for the desktop app. The source code is maintained in a separate private repository.

## Download

Visit the [Releases](https://github.com/emilstepanian/immersed-releases/releases) page to download the latest version for macOS or Windows.

## Auto-Updates

The Immersed app automatically checks for updates from this repository. You'll be notified when a new version is available and can update directly from within the app.

## Supported Platforms

- **macOS** (Apple Silicon and Intel)
- **Windows**
